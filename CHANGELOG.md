# Changelog

## 0.1.0

Initial public release.

- Added the macOS desktop app for local stem separation and mix export.
- Added local audio, video, and public YouTube URL import.
- Added batch processing, run comparison, stem gain controls, muting, and export bundles.
- Added local cleanup tools for temp files, export bundles, and old non-preferred runs.
- Added contributor docs, issue templates, security notes, and release checks.
